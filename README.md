# SaintCyrillic

**SaintCyrillic** is a Python-based web application that converts images into Cyrillic ASCII art. This project uniquely blends image processing techniques with the Cyrillic alphabet to create textual art.

## Features

- **Image Processing**: Converts images to grayscale and optionally applies filters like Gaussian blur and edge detection.
- **Cyrillic ASCII Art**: Maps grayscale values to a custom set of Cyrillic characters.
- **Web Interface**: Provides an text only web interface for uploading images and generating Cyrillic ASCII art.
